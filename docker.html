HTML to PDF





Pendant ce projet, j'ai développé avec mon équipe un outil qui permet de générer automatiquement des PDF à partir de données bruts. Lors de ce projet, nous avons utilisé de nombreux langages et outils ; HTML & CSS pour créer notre page à convertir en PDF, PhP pour la gestion des données et leur tri, le bash pour l'utilisation et la création de conteneurs Dockers qui permettent d'automatiser le processus.
